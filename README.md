## Calculadora básica
                                    # Ejemplos
- Suma (+)                          # 2+3
- Resta (-)                         # 6-5
- Multiplicación (*)                # 4*6
- División (/)                      # 8/4
- Factorial (!)                     # 5!
- Comprobación de número primo (P?) # 89 `P?`
